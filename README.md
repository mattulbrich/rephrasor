# rephrasor
Improve the language of scientific texts using the OpenAI GPT API

# Keywords:

    <rephrasor:context-file>...</>

Interpret ... as a filename and add the content to the context    
    
    <rephrasor:copy>
       ...
    </rephrasor:copy>
    
Verbatim copy of ... to the context (still valid input)
    
    <rephrasor:add> ... </>
    
Add the (single line) text ... to the context.
